🔍 TROUBLESHOOTING

Issue: "ModuleNotFoundError: No module named 'cv2'"
Solution: Install OpenCV
  pip install opencv-python

Issue: "ModuleNotFoundError: No module named 'PyQt6'"
Solution: Install PyQt6
  pip install PyQt6 PyQt6-multimedia

Issue: "ModuleNotFoundError: No module named 'mediapipe'"
Solution: Install MediaPipe
  pip install mediapipe

Issue: Camera not working
Solution: 
  1. Check system camera permissions (macOS: Settings → Security & Privacy)
  2. Grant Terminal camera access
  3. Restart the application

Issue: App won't start
Solution:
  1. Try: python3 motionbloom_run.py (direct launcher)
  2. Check for Python errors in console
  3. Verify all dependencies installed
  4. Try installing fresh: pip install -r requirements.txt --force-reinstall
