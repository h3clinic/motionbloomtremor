**Copy & paste these commands to get started:**

```bash

```

## Generate Using Replicate (Easiest)
```bash
# 1. Sign up at https://replicate.com/
# 2. Get API token, then:

export REPLICATE_API_TOKEN="your_token_here"
pip install replicate

# 3. Create my_video.py with code from examples, then:
python3 my_video.py
```

## Generate Using Wan 2.2 (Best Quality)
```bash
# 1. Install Wan 2.2
cd /tmp
git clone https://github.com/MaaS2/Wan.git
cd Wan
pip install -r requirements.txt
# Download weights (see Wan README)

# 2. Copy batch script
cp /Users/aharshi/MotionBloomAppVersion/batch_generate_tier_a.sh .

# 3. Edit WAN_DIR path, then run
nano batch_generate_tier_a.sh  # Edit WAN_DIR
bash batch_generate_tier_a.sh
```

## Validate Generated Videos
```bash
cd /Users/aharshi/MotionBloomAppVersion
mkdir -p test_videos
# Copy your video to test_videos/

python3 topotremor/ai_video_qa.py test_videos/
cat test_videos/qa_results.json
```

## Full Workflow (After Testing)

```bash
cd /Users/aharshi/MotionBloomAppVersion

# 1. Generate more videos (50-100 across all tiers)
# Use scripts/generate_videos.py or manual commands

# 2. Run QA on all
python3 topotremor/ai_video_qa.py videos/

# 3. Ingest valid ones
python3 topotremor/ai_video_ingestion.py batch \
  --video_list videos/qa_results.json \
  --output windows.json

# 4. Generate reports
python3 topotremor/ai_video_reporting.py \
  videos/qa_results.json \
  windows/ \
  reports/

# 5. Use in training with sample_weight = label_confidence
```
